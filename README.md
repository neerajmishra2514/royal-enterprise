# Royal Enterprise

Welcome to the Royal Enterprise project.

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
